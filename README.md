# EXPRESS - vite js.

## Installation

npm - yarn - pnpm

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run dev
$ pnpm run serve

# build. public/dist
$ pnpm run build

# production mode
$ pnpm run start
```
